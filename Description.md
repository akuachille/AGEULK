# AGEULK Website Description

## Overview
**AGEULK** (General Association of Students at ULK) is a modern, high-performance web platform designed to represent the student body of the Université Libre de Kigali (ULK). The website serves as a central hub for information regarding student leadership, university governance, available services, and campus life.

## Design Philosophy
The website features a **premium glassmorphism UI/UX design**, characterized by:
- **Translucent Interfaces**: Frosted glass effects that provide a modern and clean aesthetic.
- **Dynamic Interactions**: Smooth parallax scrolling and micro-animations that enhance user engagement.
- **Theme Versatility**: A built-in theme toggle allowing users to switch between a sophisticated dark mode and a clean light mode.
- **Full Responsivity**: Optimized for seamless viewing across mobile, tablet, and desktop devices.

## Key Features & Sections

### 1. Hero Experience
The landing page opens with a high-impact parallax hero section featuring the AGEULK logo and a call to action, setting a professional tone from the first interaction.

### 2. About AGEULK
Provides a clear understanding of the association's identity, mission, vision, and core values (Integrity, Collaboration, and Commitment).

### 3. Leadership Showcase
- **AGEULK Committee**: Highlights the primary student leaders with profile cards including the President, Vice President, Secretary General, and Treasurer.
- **University Governance**: Introduces the University's executive leadership, including the Vice-Chancellor and Director of Student Affairs, fostering a connection between students and administration.
- **Extended Team**: A structured data table showcasing the broader leadership team, including various ministers and committee heads.

### 4. Student Services
A dedicated section outlining the support systems provided by AGEULK:
- **Academic Support**: Advising and career development resources.
- **Club Coordination**: Management and support for diverse student organizations.
- **Campus Life**: Sports, arts, and cultural event organization.

### 5. Interactive Blog
A integrated blog system designed to keep the student body updated on the latest news, announcements, and events happening around the campus.

### 6. Seamless Contact
Integrated contact options including direct email links, phone connectivity, and a one-click WhatsApp redirection for instant communication.

## Technical Stack
- **Structure**: Semantic HTML5
- **Styling**: Modern CSS3 (Custom Properties, Flexbox, Grid, Glassmorphism techniques)
- **Logic**: Vanilla JavaScript (Scroll animations, Parallax effects, Theme management)
- **Optimization**: SEO-friendly structure and fast-loading assets.
