# AGEULK
AGEULK is the General Association of Students at ULK. This website showcases the association’s leadership, services, university partners, and contact information using a modern glassmorphism interface and interactive scrolling.

## Features
- Responsive design for mobile, tablet, and desktop devices
- Glassmorphism visuals with blurred translucent panels
- Custom smooth scrolling and parallax background effects
- Theme toggle for light and dark modes
- Navigation sections for Home, About, Leadership, Services, and Contact
- University leadership section with featured profiles

## Technologies
- HTML
- CSS
- JavaScript

## Project Structure
- `Index.html` — Main website markup and section structure
- `CSS/Index.css` — Styling for the site, responsive layout, and UI theme
- `CSS/script.js` — Interactive behavior, smooth scrolling, theme toggle, and scroll animations
- `Utilities/` — Image assets used throughout the website

## Getting Started
1. Clone this repository to your local machine.
2. Open `index.html` in your web browser.
3. Browse sections like About, Leadership, Services, and Contact.

## Deployment on GitHub Pages
This project is configured for deployment on GitHub Pages.

1. Go to your repository settings on GitHub.
2. Navigate to the "Pages" section.
3. Under "Source", select "Deploy from a branch".
4. Choose the `main` branch and `/ (root)` folder.
5. Save the settings.
6. Your site will be available at `https://kingachille-123.github.io/AGEULK/`.

Note: The `.nojekyll` file ensures GitHub Pages serves the site as static files without Jekyll processing.
